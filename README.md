# Portfolio Jurídico

Site profissional de portfólio/currículo jurídico para **Fernanda Vitória Silva**, estudante de Direito da UNIFAVIP Wyden, criado para apresentação profissional, busca por estágio e publicação em GitHub Pages.


> Versão V8: ajustes finais na hero section, com biblioteca jurídica mais aberta no fundo, card do WhatsApp reposicionado abaixo da foto e tipografia revisada para evitar cortes e textos estourando molduras.

## Objetivo

O projeto apresenta formação, experiências, habilidades, áreas de interesse e canais de contato em uma identidade visual jurídica, elegante e moderna, com foco em oportunidades de estágio.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript puro
- SVG inline para ícones
- Layout responsivo

## Funcionalidades

- Header com navegação entre seções
- Hero section premium com imagem da Fernanda e ambiente jurídico
- Botão para baixar currículo
- Botões de contato e WhatsApp com mensagem automática
- Cards de formação, experiências e habilidades
- Áreas de interesse jurídico
- Seção de contato completa
- Menu mobile
- Scroll suave
- Animações leves ao rolar a página
- Estrutura pronta para GitHub Pages

## Como rodar localmente

Abra a pasta do projeto no VS Code e use a extensão **Live Server**.

Também é possível abrir o arquivo `index.html` diretamente no navegador, pois o projeto não depende de backend.

## Como trocar a foto

Substitua o arquivo abaixo mantendo o mesmo nome:

```txt
assets/fernanda-foto.jpg
```

## Como trocar o fundo jurídico

Substitua o arquivo abaixo mantendo o mesmo nome:

```txt
assets/law-office-bg.jpg
```

## Como adicionar o currículo em PDF

Coloque o currículo dentro da pasta `assets` com o nome:

```txt
curriculo-fernanda.pdf
```

O botão **Baixar Currículo** já aponta para esse arquivo.

## WhatsApp

O botão de WhatsApp abre uma conversa com o número configurado e a mensagem automática:

```txt
Olá, vi seu portfólio jurídico e gostaria de conversar sobre uma oportunidade de estágio.
```

## Publicação no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie os arquivos do projeto.
3. Vá em **Settings > Pages**.
4. Em **Branch**, selecione `main` e a pasta `/root`.
5. Salve e aguarde o link do GitHub Pages.
